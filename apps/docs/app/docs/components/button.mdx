# Button

Displays a button or a component that looks like a button.

## Installation

```bash
npm install @radix-ui/react-slot class-variance-authority
```

## Usage

<Preview code={`import { Button } from "@/components/ui/button";

export function Example() {
  return <Button>Click me</Button>;
}`}>
  <Button>Click me</Button>
</Preview>

## Examples

### Variants

The button component comes with several variants:

<Preview code={`import { Button } from "@/components/ui/button";

export function Variants() {
  return (
    <div className="flex gap-4">
      <Button variant="default">Default</Button>
      <Button variant="destructive">Destructive</Button>
      <Button variant="outline">Outline</Button>
      <Button variant="secondary">Secondary</Button>
      <Button variant="ghost">Ghost</Button>
      <Button variant="link">Link</Button>
    </div>
  );
}`}>
  <div className="flex gap-4">
    <Button variant="default">Default</Button>
    <Button variant="destructive">Destructive</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="secondary">Secondary</Button>
    <Button variant="ghost">Ghost</Button>
    <Button variant="link">Link</Button>
  </div>
</Preview>

### With Icons

You can add icons to buttons:

<Preview code={`import { Button } from "@/components/ui/button";
import { Download, ArrowRight } from "lucide-react";

export function WithIcons() {
  return (
    <div className="flex gap-4">
      <Button>
        <Download className="mr-2 size-4" />
        Download
      </Button>
      <Button variant="outline">
        Continue
        <ArrowRight className="ml-2 size-4" />
      </Button>
    </div>
  );
}`}>
  <div className="flex gap-4">
    <Button>
      <Download className="mr-2 size-4" />
      Download
    </Button>
    <Button variant="outline">
      Continue
      <ArrowRight className="ml-2 size-4" />
    </Button>
  </div>
</Preview>

### As Child

Use the `asChild` prop to compose buttons with other components:

<Preview code={`import { Button } from "@/components/ui/button";
import Link from "next/link";

export function AsChild() {
  return (
    <Button asChild>
      <Link href="/docs">Go to Docs</Link>
    </Button>
  );
}`}>
  <Button asChild>
    <Link href="/docs">Go to Docs</Link>
  </Button>
</Preview>

### Disabled State

Buttons can be disabled:

<Preview code={`import { Button } from "@/components/ui/button";

export function Disabled() {
  return (
    <div className="flex gap-4">
      <Button disabled>Disabled</Button>
      <Button variant="outline" disabled>
        Disabled Outline
      </Button>
    </div>
  );
}`}>
  <div className="flex gap-4">
    <Button disabled>Disabled</Button>
    <Button variant="outline" disabled>
      Disabled Outline
    </Button>
  </div>
</Preview>

## API Reference

For complete API documentation, see the [API Reference](/docs/components/button/api).

## Accessibility

- Buttons are keyboard accessible
- Focus states are clearly visible
- Disabled buttons are announced to screen readers
- Buttons use semantic HTML (`<button>` element)

## Styling

Buttons use Tailwind CSS and can be customized using the `className` prop or by modifying the component's variant definitions.
